# Projet : Machine Learning - Cas de Régression

Ce projet illustre l'application de **techniques de Machine Learning supervisé** pour développer un **modèle de régression** permettant d’estimer les ventes en fonction de plusieurs variables.

## Objectif du Projet
- Développer un modèle prédictif capable d’estimer les ventes  
- Comprendre l’impact des différentes variables sur les ventes  
- Visualiser et interpréter les résultats pour soutenir la prise de décision

## Méthodologie
1. **Préparation des données** : nettoyage, gestion des valeurs manquantes et normalisation.  
2. **Sélection des features** pertinentes pour le modèle.  
3. **Division des données** en ensembles d’entraînement et de test.  
4. **Application des algorithmes de régression** : Régression Linéaire, Random Forest, ou Gradient Boosting selon le cas.  
5. **Évaluation des performances** via des métriques comme RMSE, R², MAE.  
6. **Analyse des résultats** et interprétation de l’importance des variables.

## Technologies Utilisées
- Python : Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- Jupyter Notebook pour le développement et la documentation  


## Résultat
Le notebook fournit un **modèle de régression complet**, avec visualisations et analyses, permettant de prédire les ventes et d’identifier les variables les plus influentes.
